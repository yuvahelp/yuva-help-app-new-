# Changelog

## [1.0.0] – 2026-03-22 🎉 Initial Release

### Added
- Home feed with WordPress post cards (thumbnail, title, excerpt)
- Categories: Latest Jobs, Results, Admit Cards, Government Schemes, Education News
- Article reader with featured image, full content, share & open-in-browser
- Bottom navigation: Home · Categories · Latest Updates · Search · About
- Offline cache with Room database
- Hourly background sync via WorkManager
- Local notifications on fresh content (Android 13+ permission aware)
- Retrofit + Moshi for WordPress REST API (`yuva.help/wp-json/wp/v2/posts`)
- Coil for image loading
- Material 3 theming
